Trip App

O Trip App é uma aplicação móvel desenvolvida para ajudar os usuários a planejar suas viagens. Com funcionalidades como criação de roteiros, sugestões de destinos e integração com mapas, o aplicativo oferece uma experiência completa para viajantes. Music App

O Music App é uma plataforma de streaming de música que permite aos usuários ouvir suas músicas favoritas em qualquer lugar. Desenvolvido com Swift e Node.js, o aplicativo oferece uma vasta biblioteca de músicas, playlists personalizadas e recomendações baseadas no gosto do usuário. A interface amigável e a performance otimizada garantem uma experiência de uso agradável.

Spotify App

O Spotify App é uma aplicação que replica algumas das funcionalidades do famoso serviço de streaming de música. Desenvolvido com React e Redux, o aplicativo permite aos usuários buscar músicas, criar playlists e seguir artistas. A integração com a API do Spotify garante acesso a uma vasta coleção de músicas e uma experiência de usuário fluida e responsiva.
